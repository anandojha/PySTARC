# Barnase-barstar (protein-protein complex)

## System

Barnase-barstar is a classic Brownian dynamics benchmark. Association is driven by electrostatic steering between positively charged barnase (ribonuclease) and negatively charged barstar (inhibitor). The wild-type and R59A mutant are provided to validate absolute k<sub>on</sub> and the effect of a charge mutation on the association rate.

| Property               | WT                                     | R59A                                   |
|------------------------|----------------------------------------|----------------------------------------|
| PDB                    | 1BRS (chains A + D)                    | 1BRS (chains A + D), ARG59 mutated     |
| Receptor               | Barnase (1701 atoms, Q = +2 e)         | Barnase (1687 atoms, Q = +1 e)         |
| Ligand                 | Barstar (1404 atoms, Q = -5 e)         | Barstar (1404 atoms, Q = -5 e)         |
| b-surface              | 80.0 Angstrom                          | 80.0 Angstrom                          |
| Escape sphere          | 160.0 Angstrom                         | 160.0 Angstrom                         |
| Debye length           | 13.6 Angstrom (50 mM NaCl)            | 13.6 Angstrom (50 mM NaCl)            |
| Reaction pairs         | 2 (ARG81-ASP147, ARG57-GLU182)         | 1 (ARG81-ASP147)                       |
| Reaction cutoff        | 7.0 Angstrom                           | 7.0 Angstrom                           |
| n_needed               | 1                                      | 1                                      |
| max_dt                 | 100.0 ps                               | 100.0 ps                               |
| Born desolvation       | enabled                                | enabled                                |
| Hydrodynamic radii     | auto (receptor 18.3, ligand 17.0)      | auto                                   |
| Trajectories           | 100,000                                | 100,000                                |
| Experimental k<sub>on</sub> | 3-6 x 10<sup>8</sup> M<sup>-1</sup>s<sup>-1</sup> | ~6.5 x 10<sup>7</sup> M<sup>-1</sup>s<sup>-1</sup> |

## Directory structure

```
barnase_barstar/
  WT/           Wild-type barnase-barstar
  R59A/         R59A barnase mutant (one fewer positive charge)
```

Each subdirectory is self-contained. All files needed to run the simulation are pre-generated and included so that `run.sh` works out of the box without running `setup.py` first. Running `setup.py` regenerates them from scratch.

## Files

| File                | Provided | Generated by setup.py | Description                                                                 |
|---------------------|----------|-----------------------|-----------------------------------------------------------------------------|
| `1BRS.pdb`          | yes      | yes (downloaded)      | Crystal structure from the PDB.                                             |
| `setup.py`          | yes      |                       | Automated setup: downloads PDB, extracts chains, parameterizes, splits PQR, writes reaction criterion and input files. |
| `run.sh`            | yes      |                       | Runs the BD simulation.                                                     |
| `complex.pdb`       | yes      | yes                   | Clean PDB from tleap with hydrogens added and residues renumbered.          |
| `complex.prmtop`    | yes      | yes                   | AMBER topology (partial charges, atom types, connectivity).                 |
| `complex.rst7`      | yes      | yes                   | AMBER coordinate file matching the topology.                                |
| `complex.pqr`       | yes      | yes                   | Full complex PQR from ambpdb.                                               |
| `receptor.pqr`      | yes      | yes                   | Receptor (barnase) PQR, split by residue number (residues 1-108).           |
| `ligand.pqr`        | yes      | yes                   | Ligand (barstar) PQR, split by residue number (residues 109+).              |
| `rxns.xml`          | yes      | yes                   | Reaction criterion with atom pairs and cutoff distances.                    |
| `input.xml`         | yes      | yes                   | PySTARC input file with all simulation parameters.                          |

## Setup and run

All generated files are included, so setup is optional. To run directly:

```bash
conda activate PySTARC
module load cuda
cd examples/barnase_barstar/WT
bash run.sh
```

For the R59A mutant:

```bash
cd examples/barnase_barstar/R59A
bash run.sh
```

To regenerate all files from scratch (optional):

```bash
cd examples/barnase_barstar/WT
python setup.py
python ../../../run_pystarc.py input.xml
```

## Output files

After the simulation completes, all results are written to `bd_sims/`.

| Output file              | Description                                                                                                                    |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| `results.json`           | k<sub>on</sub>, P<sub>rxn</sub>, Wilson 95% CI, k<sub>b</sub>, D<sub>rel</sub>, wall time, and GPU info.                      |
| `convergence.json`       | Convergence analysis: SE, relative SE, Wilson CI, convergence verdict, and trajectory estimates for target precision.           |
| `trajectories.csv`       | Per-trajectory record: number of steps, starting pose, minimum distance reached, and number of returns from the escape sphere. |
| `encounters.csv`         | Binding encounter poses for reacted trajectories: final position, orientation, and contact distances.                          |
| `near_misses.csv`        | Trajectories that approached the reaction surface but escaped.                                                                 |
| `fpt_distribution.csv`   | First-passage times for reacted trajectories.                                                                                  |
| `pose_clusters.csv`      | Clustered binding poses from encounter geometries.                                                                             |
| `paths.npz`              | Full trajectory coordinates sampled at configurable intervals.                                                                 |
| `energetics.npz`         | Per-step energies and forces along trajectories.                                                                               |
| `radial_density.csv`     | Radial probability density as a function of distance from the receptor.                                                        |
| `angular_map.npz`        | Angular occupancy map (theta, phi) on the b-surface.                                                                           |
| `contact_frequency.csv`  | Per-pair contact frequencies for the reaction criterion atom pairs.                                                            |
| `milestone_flux.csv`     | Net flux across radial shells.                                                                                                 |
| `transition_matrix.npz`  | Radial shell-to-shell transition counts.                                                                                       |
| `p_commit.npz`           | Commitment probabilities at each radial shell.                                                                                 |

A timestamped log file (`pystarc_YYYYMMDD_HHMMSS.log`) is also written to `bd_sims/` containing the full simulation output.

## Reaction criterion

The reaction criterion follows Gabdoulline and Wade (1997). Two intermolecular charge-charge contacts define the binding interface:

| Pair | Receptor atom            | Ligand atom             | Paper residues   |
|------|--------------------------|-------------------------|------------------|
| 1    | ARG81 NH2 (barnase)      | ASP147 OD1 (barstar)    | R83-D35          |
| 2    | ARG57 NH1 (barnase)      | GLU182 OE1 (barstar)    | R59-E76          |

Residue numbering differs from the paper because tleap renumbers both chains sequentially (barnase 1-108, barstar 109-195). With `n_needed=1`, a single contact firing is sufficient to count a reaction. The R59A mutant has only pair 1 because the ARG57 side chain is removed by the mutation.

## Notes

- Unlike the protein-ligand examples, barnase-barstar is a protein-protein system. Both molecules are standard amino acids, so splitting is done by residue number rather than residue name.
- The `max_dt=100.0` parameter caps the adaptive timestep. This is critical for protein-protein systems where the b-surface is far from the binding site. Without this cap, the adaptive timestep grows too large at far separations and trajectories overshoot the reaction region.
- The R59A mutant setup.py applies the mutation by stripping ARG59 side-chain atoms beyond CB and relabeling the residue as ALA before tleap parameterization.
- PQR files are generated from the AMBER topology via `cpptraj` and `ambpdb`.
