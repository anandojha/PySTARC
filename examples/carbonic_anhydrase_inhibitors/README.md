# Carbonic anhydrase sulfonamide inhibitors (protein-ligand)

## System

Seven sulfonamide inhibitors binding three carbonic anhydrase isozymes (CA XIII, CA II, CA I). All ligands are deprotonated at the sulfonamide nitrogen (net charge = -1), which is the binding-competent form. BD models the diffusion-limited encounter of this deprotonated species with the Zn-containing active site.

| System | Isozyme | PDB | Ligand | Exp k<sub>on</sub> (M<sup>-1</sup>s<sup>-1</sup>) |
|--------|---------|-----|--------|-----------------------------------------------------|
| `ca13_azm` | CA XIII | 3CZV | Acetazolamide (AZM) | 1.5 x 10<sup>6</sup> |
| `ca13_vd1125` | CA XIII | 3CZV | VD11-25 | 4.6 x 10<sup>5</sup> |
| `ca13_vd1126` | CA XIII | 3CZV | VD11-26 | 1.5 x 10<sup>6</sup> |
| `ca13_vd1209` | CA XIII | 3CZV | VD12-09 | 3.3 x 10<sup>5</sup> |
| `ca13_vd1269` | CA XIII | 3CZV | VD12-69-1 | 2.5 x 10<sup>6</sup> |
| `ca1_vd1269` | CA I | 2NMX | VD12-69-1 | 2.7 x 10<sup>6</sup> |
| `ca2_vd1142` | CA II | 3HS4 | VD11-4-2 | 1.8 x 10<sup>6</sup> |

Experimental k<sub>on</sub> values are intrinsic association rates from Linkuviene et al., J. Med. Chem. 2018.

## Shared parameters

| Property               | Value                                |
|------------------------|--------------------------------------|
| Ligand charge          | -1 e (deprotonated sulfonamide)      |
| b-surface              | 60.0 Angstrom                        |
| Escape sphere          | 120.0 Angstrom                       |
| Debye length           | 9.62 Angstrom (100 mM NaCl)         |
| Reaction cutoff        | 5.0 Angstrom                         |
| Reaction pairs         | 2 (THR199 OG1 - sulfonamide N, GLU106 OE1 - amide N) |
| n_needed               | 2                                    |
| Born desolvation       | enabled                              |
| Hydrodynamic radii     | auto                                 |
| Trajectories           | 100,000                              |

  ca1_vd1269/       CA I + VD12-69-1

Each subdirectory is self-contained. All files needed to run the simulation are pre-generated and included so that `run.sh` works out of the box without running `setup.py` first. Running `setup.py` regenerates them from scratch.

## Files

Each subdirectory contains:

| File                | Provided | Generated by setup.py | Description                                                                 |
|---------------------|----------|-----------------------|-----------------------------------------------------------------------------|

| `setup.py`          | yes      |                       | Automated setup: downloads PDB, extracts protein + Zn, generates deprotonated ligand from SMILES via rdkit/obabel, parameterizes with ff14SB (protein) and GAFF2/AM1-BCC (ligand), generates PQR files, reaction criterion, and input files. |
| `run.sh`            | yes      |                       | Runs the BD simulation.                                                     |
| `receptor.pdb`      | yes      | yes                   | Clean receptor PDB from tleap (protein + Zn).                               |
| `receptor.pqr`      | yes      | yes                   | Receptor PQR from ambpdb.                                                   |
| `ligand.pdb`        | yes      | yes                   | Clean ligand PDB from tleap.                                                |

| `protein.prmtop`    | yes      | yes                   | Receptor AMBER topology.                                                    |

## Setup and run

All generated files are included, so setup is optional. To run a single system:

```bash
conda activate PySTARC
module load cuda
cd examples/carbonic_anhydrase_inhibitors/ca13_azm
bash run.sh
```

To run all 7 systems sequentially:

```bash
conda activate PySTARC
module load cuda
cd examples/carbonic_anhydrase_inhibitors
for d in ca13_azm ca13_vd1125 ca13_vd1126 ca13_vd1209 ca13_vd1269 ca1_vd1269 ca2_vd1142; do
    cd $d && bash run.sh && cd ..
done
```

To regenerate all files from scratch (requires rdkit and openbabel):

```bash
cd examples/carbonic_anhydrase_inhibitors/ca13_azm
python setup.py
python ../../../run_pystarc.py input.xml
```

## Dependencies

In addition to the standard PySTARC dependencies, setup.py requires rdkit and openbabel (both installed via `conda install -c conda-forge rdkit openbabel`) to generate the deprotonated sulfonamide ligand from SMILES. Antechamber (from AmberTools) is used for GAFF2 parameterization with AM1-BCC partial charges.

## Output files

After the simulation completes, all results are written to `bd_sims/` within each system directory.

| Output file              | Description                                                                                                                    |
|--------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| `results.json`           | k<sub>on</sub>, P<sub>rxn</sub>, Wilson 95% CI, k<sub>b</sub>, D<sub>rel</sub>, wall time, and GPU info.                      |
| `convergence.json`       | Convergence analysis: SE, relative SE, Wilson CI, convergence verdict, and trajectory estimates for target precision.           |
| `trajectories.csv`       | Per-trajectory record: number of steps, starting pose, minimum distance reached, and number of returns from the escape sphere. |
| `encounters.csv`         | Binding encounter poses for reacted trajectories: final position, orientation, and contact distances.                          |
| `near_misses.csv`        | Trajectories that approached the reaction surface but escaped.                                                                 |
| `fpt_distribution.csv`   | First-passage times for reacted trajectories.                                                                                  |
| `pose_clusters.csv`      | Clustered binding poses from encounter geometries.                                                                             |
| `paths.npz`              | Full trajectory coordinates sampled at configurable intervals.                                                                 |
| `energetics.npz`         | Per-step energies and forces along trajectories.                                                                               |
| `radial_density.csv`     | Radial probability density as a function of distance from the receptor.                                                        |
| `angular_map.npz`        | Angular occupancy map (theta, phi) on the b-surface.                                                                           |
| `contact_frequency.csv`  | Per-pair contact frequencies for the reaction criterion atom pairs.                                                            |
| `milestone_flux.csv`     | Net flux across radial shells.                                                                                                 |
| `transition_matrix.npz`  | Radial shell-to-shell transition counts.                                                                                       |
| `p_commit.npz`           | Commitment probabilities at each radial shell.                                                                                 |

A timestamped log file (`pystarc_YYYYMMDD_HHMMSS.log`) is also written to `bd_sims/` containing the full simulation output.

## Reaction criterion

Two crystal-structure contacts between the protein active site and the sulfonamide define the reaction criterion:

| Pair | Receptor atom                  | Ligand atom      | Contact type                  |
|------|--------------------------------|------------------|-------------------------------|
| 1    | THR199 OG1 (gatekeeper)        | Sulfonamide N    | Zn-coordinating H-bond        |
| 2    | GLU106 OE1 (proton shuttle)    | Amide N          | Relay H-bond                  |

Residue numbers are after tleap renumbering (offset -3 from PDB numbering). With `n_needed=2`, both contacts must be satisfied simultaneously. The 5.0 Angstrom cutoff reflects the tight active-site geometry of carbonic anhydrases.

## Notes

- All ligands are deprotonated sulfonamides (charge = -1). At physiological pH, only ~0.1% of the sulfonamide exists in this binding-competent form. The experimental intrinsic k<sub>on</sub> (corrected for protonation equilibrium) is the correct comparison target for BD.
- The receptor includes the active-site Zn<sup>2+</sup> ion (loaded via `frcmod.ions234lm_126_tip3p` in tleap).
- CA XIII (PDB 3CZV) is used as the reference structure for 5 of 7 systems. The same protein structure is paired with different ligands generated from SMILES.
- CA I (PDB 2NMX) and CA II (PDB 3HS4) are different isozymes with distinct active-site electrostatics, allowing cross-isozyme comparison.
- PQR files are generated from the AMBER topology via `cpptraj` and `ambpdb`.
